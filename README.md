# pratice-for-node-v8.6.0
一个node项目，node运行版本v8.6.0

### 注意事项
> 该项目在window系统下开发，所以有些npm run命令在linux系统下会报错


### 克隆项目
```
git clone git@github.com:wlybq/pratice-for-node-v8.6.0.git
```

### 下载依赖
```
npm install
```

### 编译
```
npm run compile(第一次编译)
```

### 自动编译
```
npm run watch(第一次编译后才可用)
```

### 运行
```
npm run dev
```
